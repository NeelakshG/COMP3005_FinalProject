## Installation
1. Install PostgreSQL if not already installed
2. Install IntelliJ if not already installed

```bash
git clone https://github.com/celinaayangg/COMP3005_FinalProject.git
cd COMP3005_FinalProject
```

## Usage
1. Start up pgAdmin and connect to make sure database is running
2. Run the initialized DDL.sql and DML.sql into pgAdmin to populate
3. Modify DBConnection class to connect your database based on personal user credentials
4. Run the Main.java program
